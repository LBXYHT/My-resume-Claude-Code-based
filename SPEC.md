# 个人简历网站规范文档

## 1. 项目概述
- **项目名称**: Minimal Resume
- **项目类型**: 单页个人简历网站
- **核心功能**: 展示个人信息、技能、项目经验和联系方式
- **目标用户**: 招聘方、业务合作伙伴

## 2. UI/UX 规范

### 布局结构
- **整体**: 单页垂直滚动布局
- **头部**: 固定导航栏（透明背景，滚动后变实色）
- **区块**:
  - Hero 区域（全屏高度，姓名+标题）
  - 关于我/基本信息
  - 技能展示
  - 项目展示
  - 联系方式
  - 页脚

### 响应式断点
- 移动端: < 768px
- 平板: 768px - 1024px
- 桌面: > 1024px

### 视觉设计

#### 配色方案
- **主背景**: #FAFAFA（极浅灰）
- **次背景**: #FFFFFF（纯白卡片）
- **主文字**: #1A1A1A（近黑）
- **次文字**: #6B6B6B（中性灰）
- **强调色**: #2D2D2D（深灰）
- **边框/分割线**: #E5E5E5
- **悬停态**: #000000

#### 字体
- **标题**: "Playfair Display", serif（优雅衬线体）
- **正文**: "DM Sans", sans-serif（现代无衬线）
- **字号**:
  - Hero 标题: 4rem (桌面) / 2.5rem (移动)
  - 区块标题: 2rem
  - 正文: 1rem
  - 小字: 0.875rem

#### 间距系统
- 区块间距: 120px（桌面）/ 80px（移动）
- 卡片内边距: 40px
- 元素间距: 24px / 16px / 8px

#### 视觉效果
- 卡片阴影: 0 4px 24px rgba(0,0,0,0.06)
- 悬停阴影: 0 8px 32px rgba(0,0,0,0.1)
- 过渡动画: all 0.4s cubic-bezier(0.4, 0, 0.2, 1)

### 组件规范

#### 导航栏
- 高度: 80px
- Logo/姓名居左，导航链接居右
- 移动端: 隐藏菜单，汉堡按钮
- 滚动后: 背景 #FFFFFF，底部阴影

#### Hero 区域
- 全屏高度 (100vh)
- 姓名 + 职业标题 + 一句话简介
- 向下滚动提示（动画箭头）

#### 基本信息卡片
- 左侧头像（圆形，120px）
- 右侧信息（姓名、职位、简介）
- 信息列表（邮箱、电话、位置）

#### 技能展示
- 网格布局（3列桌面 / 2列平板 / 1列移动）
- 技能标签样式：圆角矩形，灰底黑字
- 技能分组展示

#### 项目卡片
- 卡片式布局
- 项目名称 + 描述 + 技术栈 + 链接
- 悬停效果：轻微上浮 + 阴影加深

#### 联系方式
- 居中布局
- 大字邮箱/电话
- 社交图标链接

## 3. 功能规范

### 核心功能
1. **平滑滚动导航**: 点击导航跳转到对应区块
2. **滚动动画**: 区块进入视口时淡入上滑
3. **移动端菜单**: 汉堡按钮展开/收起导航
4. **导航栏滚动响应**: 滚动超过50px后改变样式

### 交互行为
- 导航链接 hover: 颜色变黑，下划线出现
- 卡片 hover: translateY(-4px) + 阴影加深
- 按钮/链接: 0.3s 过渡效果
- 滚动动画: 使用 Intersection Observer
